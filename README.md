# flask_structre

# Two structres
## app 
```
apps
├── app.py
├── conf
│   ├── dev_config.py
│   └── test_config.py
└── project
    ├── __init__.py
    ├── db.py
    ├── settings.py
    └── users
        ├── __init__.py
        ├── models.py
        ├── route.py
        └── templates
```

## funtion
```
functions
├── app.py
├── conf
│   ├── dev_config.py
│   └── test_config.py
└── project
    ├── __init__.py
    ├── __init__.pyc
    ├── models
    │   ├── __init__.py
    │   └── base.py
    ├── routes
    │   ├── __init__.py
    │   └── users.py
    ├── settings.py
    └── templates
        └── base.html
```

`reference： https://lepture.com/en/2018/structure-of-a-flask-project`
